Info

Moderation is the main bot in the server. It is responsbile for logging all activities such as user joins, user role changes and user bans.
  

Bot Prefix

The prefix of the bot is an exclamation mark !

activity [user] ([--global])

Gets the number of messages sent by a user in the last 30 days, either server only or globally.

Your activity - ie. !activity
Another user's activity - ie. !activity @T-Marey

Your activity globally - ie. !activity --global
Another user's activity globally - ie. !activity @Callofduty4 --global

Command restriction: none.

ban [user] [reason]

Bans a user.

ie. !ban @343TheGuiltyProphet being guilty

Command restriction: Wiki Moderators, Wiki Administators.
 

kick [user] [reason]

Kicks a user from the server. It also sends them a message with the reason.

ie. !kick @Tem being a foo

Command restriction: Wiki Moderators, Wiki Administators.
  

mute [user] ([length];) [reason]

Mutes a user, making them unable to post any messages, whilst still remaining in the server.
By default, the mute length is one hour.

Mute a user for one hour - ie. !mute @Shad discussing TF2.

Mute a user for one day - ie. !mute @Shad 1d; discussing TF2.
Note: The semi-colon ; must be included. It is used to separate the time length from the reason.

Time lengths: seconds = s, minutes = m, hour = h, day = d.

Command restriction: Wiki Moderators, Wiki Administators.
  

prune [no of msgs] [user(s)]

Deletes the specified number of messages. The maximum is 100 per command.

Delete 99 messages from a single user - !prune 99 @T-Marey

Delete 15 messages from multiple users - !prune 15 @HassieSwift621 @T-Marey
Note: There is no separator to separate the list of users.

Command restriction: Wiki Moderators, Wiki Administators.

searchlogs [parameters]

Search the chat log.

Search logs without a specifc user, searchlogs all [parameters] - ie. !searchlogs all --find:'hi'

Search for a specific user in the logs, searchlogs [user] [parameters] - ie. !searchlogs @TheCountess --find:'hi'


Parameters
  • --latest - This will order the log search from latest message to oldest message.
  • --find:'parameter' - Find a specific word/phrase.
    Note: The quotation mark ' must be included.
  • --show:# - Only show a specific number of search results. The max is 30 per page.
  • --ignoreself - This removes your messages from the search results.
  • --here - Only search your current channel.
  • --ids - Show the user IDs and message IDs.
  • --page:# - Choose search results page number

Note: No separator is used to separate each search parameter. Instead a double dash -- is used to declare each parameter.


Command restriction: Wiki Administators.

autorole [role(s)]

Automatically set roles when a user joins the server.

Set the role to unverified and users - ie. !settings autorole unverified users
Note: No separator is used to separate the list of roles.

Turn off auto role - ie. !settings autorole off.

Command restriction: Wiki Administators.

grant ["role"] [permission]

Grants a role a permission.

ie. !grant "b'crats" prune

Grantable permissions:
  • administrator - Gives the role access to all parts of the bot.
  • ban - Allows the role to ban a user with the ban command.
  • changemodlogs - Allows the role to modify the moderation logs, ie. changing the ban reason.
  • grant - Allows the role to grant/revoke these permissions to roles.
  • kick - Allows the role to kick a user with the kick command.
  • logs - Gives the role to view the chat logs.
  • modlogs - Gives the role access to the moderation logs.
  • mute - Allows the role to mute a user with the mute command.
  • override - Allows the role to override specific channel permissions, ie. using bot commands in a channel where the bot is disabled.
  • prune - Allows the role to delete messages with the prune command.
  • resolve - Allows the role to remove a moderation log.
  • roleadd - Allows the role to add a role to a user.
  • roleremove - Allows the role to remove a role from a user.
  • unmute - Allows the role to unmute a user with the un-mute command.
  • warn - Allows the role to warn a user with the warn command.

Command restriction: Wiki Administators.
  

grantable ["role"] [yes|no]

Set a role grantable or un-grantable.

Set a role un-grantable - ie. !grantable "mods" no

Command restriction: Wiki Administators.
  

ignore [parameter] [channel(s)]

Manage channels with the bot commands ignored.

Set channel "main" to ignore bot commands - ie. !settings ignore add #main

Set channel "main" and "wiki-discussion" to ignore bot commands and overwrite previous settings - ie. !settings ignore set #main #wiki-discussion

Remove channel "main" from ignoring bot commands - ie. !settings ignore remove main

List channels where bot commands are ignored - ie. !settings ignore list

Note: There is no separator used to separate the list of channels.

Command restriction: Wiki Administators.
  

join [message]

Sets the join message for a new user.

ie. !settings join Welcome {user} to the official My Little Pony Discord server. We now have {members} in our server.
Note: {user} automatically subsitutes the username. {members} automatically subsitutes the total number of users in the server.

Command restriction: Wiki Administators.

loglevel [level]

Sets the logging level.

ie. !settings loglevel 2

  • 1 - Bot commands only

Note: The default log level is 1.
